Auto-Battler Bot (Python)
Простой кликер для автоматизации боев в автобаттлере. Бот распознает кнопки на экране с помощью компьютерного зрения и нажимает их в зависимости от ситуации (бой, награда, повышение уровня).
🚀 Функционал

* Автоматический бой: Нажимает "В бой" и подтверждает вход.
* Сбор наград: Находит кнопку "Забрать" и закрывает окна после боя.
* Распознавание Уровня: Умеет отличать обычную кнопку "Продолжить" от окна повышения уровня (ищет слово "Уровень") и делает увеличенную паузу для анимации.
* Защита от ошибок: Игнорирует временные окна и ошибки поиска через try-except.

🛠 Требования
Для работы бота нужен Python 3.x и библиотека PyAutoGUI (с поддержкой OpenCV для точности).

pip install pyautogui opencv-python pillow

📸 Настройка скриншотов
Для корректной работы в папке со скриптом должны лежать следующие изображения (в формате .png):

   1. v_boy.png — основная кнопка входа в бой.
   2. v_boy_2.png — подтверждение входа в бой.
   3. prodolzhit.png — кнопка после завершения сражения.
   4. slovo_uroven.png — уникальный фрагмент текста "Уровень" (чтобы бот понимал, что нужно подождать анимацию наград).
   5. zabrat.png — кнопка получения призов.
   6. click_to_continue.png — для пропуска диалогов или заставок.
   7. otkazatsya.png — если нужно закрыть окно прерванной серии.

Важно: Делайте скриншоты именно в том разрешении экрана, в котором запускаете игру.

⚙️ Настройки кода
Внутри файла bot.py можно подправить:

* CONFIDENCE: точность совпадения (0.7 = 70%). Если бот "тупит", понизь до 0.6. Если кликает не туда — повысь до 0.8.
* DELAY: общая задержка между циклами проверки экрана.

⚠️ Запуск и остановка

   1. Запусти скрипт: python bot.py
   2. Переключись на окно с игрой.
   3. Чтобы остановить бота, нажми Ctrl + C в окне терминала/консоли или резко переведи курсор мыши в левый верхний угол экрана (функция Fail-Safe).
