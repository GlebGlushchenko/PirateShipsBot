# Battle Bot (Auto-Battler)

Бот для автоматизации боёв в автобаттлере: распознаёт кнопки на экране и нажимает их. Поддержка уведомлений в Telegram (статистика раз в час, каждая 10-я победа, предупреждение о зависании).

---

## Что умеет бот

- **Автобой** — нажимает «В бой» и подтверждение входа.
- **Сбор наград** — находит «Забрать», закрывает окна после боя.
- **Повышение уровня** — отличает обычное «Продолжить» от окна уровня (ищет «Уровень»), делает паузу под анимацию.
- **Отказ** — нажимает «Отказаться» при прерванной серии.
- **Статистика** — считает победы/поражения, золото, уровни; отправляет отчёты в Telegram.

---

## Требования

- Python 3.9+
- Библиотеки перечислены в **requirements.txt**

**Установка одной командой:**

```bash
pip install -r requirements.txt
```

На Windows можно просто запустить **setup.bat** (двойной клик) — он сам поставит все библиотеки из `requirements.txt`.

## Настройка

### 1. Конфиг — `config.py`

- **BOT** — точность `confidence`, задержки, интервал статистики (по умолчанию раз в час), проверка зависания (сек), уведомление каждые N побед.
- **TELEGRAM_CONFIG** — `token` и `chat_id` (получить у [@BotFather](https://t.me/BotFather) и [@userinfobot](https://t.me/userinfobot)), включение уведомлений и ежедневного отчёта.

### 2. Картинки — папка `images/`

Скриншоты кнопок в **том же разрешении**, в котором играешь. Формат PNG.

| Файл | Назначение |
|------|------------|
| `v_boy.png` | Кнопка «В бой» (основная) |
| `v_boy_2.png` | Подтверждение входа в бой |
| `prodolzhit.png` | «Продолжить» после боя |
| `slovo_uroven.png` | Фрагмент текста «Уровень» (для паузы при левеле) |
| `zabrat.png` | «Забрать» награду |
| `click_to_continue.png` | Пропуск диалогов/заставок |
| `otkazatsya.png` | «Отказаться» |
| `win.png` | Экран победы |
| `lose.png` | Экран поражения |

## Запуск

| Способ | Файл | Описание |
|--------|------|----------|
| С консолью | **run_bot.bat** | Двойной клик — открывается окно с логом. Остановка: закрыть окно или Ctrl+C. |
| Без окна | **Start_Bot_NoConsole.vbs** | Двойной клик — бот работает в фоне, консоль не показывается. |
| Остановка фона | **Stop_Bot.bat** | Двойной клик — завершает бота, запущенного через VBS. |

Или из терминала:

```bash
py -3.9 game_bot_with_telegram.py
```

## Уведомления в Telegram

- **Раз в час** — полная статистика (бои, победы/поражения, золото, прогресс).
- **Каждая 10-я победа** — короткое сообщение со счётчиком и золотом.
- **Зависание** — если 20 секунд нет ни одного действия, приходит предупреждение.
- **Ежедневный отчёт** — в 23:00 (если включён в `config.py`).
- **При остановке бота** — итоговое сообщение в Telegram.

Пороги (час, 10 побед, 20 сек) настраиваются в `config.py` в секции **BOT**.

## Безопасность

- Токен и chat_id хранятся в `config.py`. Не выкладывай этот файл в открытый доступ.
- PyAutoGUI по умолчанию останавливается при движении мыши в левый верхний угол экрана (Fail-Safe).
